NAME
----
babeltrace-plugin-filter-utils.muxer - Babeltrace's muxer utility
filter component class


DESCRIPTION
-----------
The Babeltrace `utils.muxer` filter component class, member of the
man:babeltrace-plugin-utils(7) plugin, once instantiated, multiplexes
the events of multiple input connections into a linear sequence of
events ordered by time on its output connection.

----
... -.
      \ +--------------------+
... --->| utils.muxer filter |---> ...
      / +--------------------+
... -'
----

The events and other notifications are untouched: the `utils.muxer`
filter component only sorts them.

A `utils.muxer` filter component's iterator signals the end of the
iteration when all its input iterators are also done.


INITIALIZATION PARAMETERS
-------------------------
The following parameters are optional unless indicated otherwise.

IMPORTANT: TODO.


PORTS
-----
A `utils.muxer` filter component only has a default input port and a
