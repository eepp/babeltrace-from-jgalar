NAME
----
babeltrace-plugin-filter-utils.trimmer - Babeltrace's trimmer utility
filter component class


DESCRIPTION
-----------
The Babeltrace `utils.trimmer` filter component class, member of the
man:babeltrace-plugin-utils(7) plugin, once instantiated, discards all
the received events with a timestamp less than a given beginning
timestamp and greater than a given end timestamp. It effectively
``cuts'', or trims traces.

----
        +----------------------+
... --->| utils.trimmer filter |---> ...
        +----------------------+
----

A `utils.trimmer` filter component modifies the `timestamp_begin` and
`timestamp_end` fields of the packet contexts it encounters to match the
beggining and end times of the trimming range when needed.


INITIALIZATION PARAMETERS
-------------------------
param:--clock-gmt=`yes` (boolean)::
	Set the time zone of the param:begin and param:end parameters
	to GMT instead of the local time zone.

param:begin='BEGIN' (string)::
    Set the trimmer's beginning time to 'BEGIN'. See
    <<time-param-fmt,Time parameter format>> for the format of 'BEGIN'.
    If this is not specified, no events are discarded until the end
    of the trimming range.

param:end='END' (string)::
    Set the trimmer's end time to 'END'. See
    <<time-param-fmt,Time parameter format>> for the format of 'END'.
    If this is not specified, no events are discarded from the beginning
    of the trimming range to the end of all the component's connections.


[[time-param-fmt]]
Time parameter format
~~~~~~~~~~~~~~~~~~~~~
The format of the param:begin and param:end parameters is:

[verse]
$$[$$__YYYY__-__MM__-__DD__ [__hh__:__mm__:]]__ss__[.__nnnnnnnnn__]

'YYYY'::
    4-digit year.

'MM'::
    2-digit month (January is `01`).

'DD'::
    2-digit day.

'hh'::
    2-digit hour (24-hour format).

'mm'::
    2-digit minute.

'ss'::
    2-digit second.

'nnnnnnnnn'::
    9-digit nanosecond.


PORTS
-----
A `utils.muxer` filter component only has a default input port and a
default output port.


QUERY OBJECTS
-------------
This component class has no objects to query.
